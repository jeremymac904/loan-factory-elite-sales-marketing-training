# Email Template Style Guide

**Format standard for all Loan Factory emails** (coaching emails, email lesson summaries, training emails, weekly recaps).
**Inherits:** `LOAN_FACTORY_BRAND_OUTPUT_RULES.md`. Read that first.
**Internal Loan Factory use only — emails go to approved Loan Factory users.**

---

## 1. Structure
```
Subject: <clear, specific, no hype — promises a concrete takeaway>
Preview: <one line that earns the open>

Hi <first name / team>,

<Hook: 1–2 sentences — the situation or the win.>

<Body: one core idea. Short paragraphs (1–3 lines) and/or a tight bullet list.>

Your next step: <one clear action>

<Sign-off>
Jeremy McDonald
The Mortgage Mentor · Loan Factory — LO Development
```

## 2. Subject line rules
- Specific and useful: "Your Power Hour script for this week," "3 follow-ups that book apps."
- No hype, no income claims, no all-caps, minimal emojis (0–1).
- 4–8 words ideal.

## 3. Voice & formatting
- Practical, supportive, direct, conversational. Speak LO-to-LO.
- Short paragraphs; bold the single most important line; one CTA.
- Plain, paste-ready text (works in Gmail/Google Chat). If HTML, use orange `#F26A1F` only for the CTA button (white text, 8px radius) and links; black `#111111` body text on white.
- Keep it skimmable in under 30 seconds.

## 4. Per-program tone
- **LO Mastery:** practical, encouraging, execution-focused. Tie to the weekly routine/scorecard.
- **Alliance:** premium, strategic, leadership voice. Tie to mastermind/advanced execution.
- **Sales & Marketing 101–601:** Jeremy-style, broker-first lesson recap. Free training tone — never sold as paid coaching.
- **AI Advantage:** forward-thinking, beginner-friendly, one tool/workflow per email.

## 5. Required elements
- Audience-appropriate greeting; one core idea; **one** clear CTA; concrete next step.
- Signature with Jeremy / The Mortgage Mentor / Loan Factory — LO Development.
- "Internal Loan Factory use only" footer line when forwarded as a template/handout.

## 6. Don'ts
- No income guarantees, manifestation language, or "borrow to join."
- No "public"/customer framing — these are approved Loan Factory users.
- No deprecated paid-coaching names. Use LO / TERA / Netlify correctly.

## 7. Production note
Output as paste-ready plain text by default. Provide subject + preview + body + sign-off. Run the Pre-Send Brand Checklist.
