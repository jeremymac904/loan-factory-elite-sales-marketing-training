# Slide Deck Style Guide

**Format standard for all Loan Factory slide decks** (LO Mastery decks, Alliance decks, 101–601 lesson decks, AI lesson decks, mastermind decks).
**Inherits:** `LOAN_FACTORY_BRAND_OUTPUT_RULES.md`. Read that first.
**Internal Loan Factory use only.**

---

## 1. Canvas
- **Aspect:** 16:9.
- **Two background modes:** Black `#000000` (cover, section breaks, big statements) and White `#FFFFFF` (content slides). Orange `#F26A1F` only as accent.
- **Font:** Arial / Helvetica Neue. Headline 32–44pt bold; body 18–24pt; captions 12–14pt `#5F6368`.

## 2. Slide types
- **Cover:** black bg, logo top-left, title (white, bold), subtitle (`#E5E7EB`), audience line + "Internal Loan Factory use only" (small, `#5F6368`), one orange accent.
- **Agenda / roadmap:** numbered list, orange numerals.
- **Content:** white bg, H1 top, ≤5 bullets OR one visual. Orange bullet markers.
- **Section break:** black bg, big orange section number + title.
- **Stat / principle:** one large number or quote, orange emphasis.
- **CTA / next steps:** orange CTA "button" block + numbered next steps.
- **Closing:** logo, attribution, "Internal Loan Factory use only."

## 3. Layout rules
- **One idea per slide.** If it needs paragraphs, it's a handout, not a slide.
- Max ~6 lines of text per slide; headline + supporting points.
- Consistent top-left H1 placement; consistent footer.
- Generous margins; align to a simple grid; never crowd.

## 4. Color usage
| Element | Color |
|---------|-------|
| Content background | White `#FFFFFF` |
| Cover/section background | Black `#000000` |
| Headline (on white) | `#111111` |
| Headline (on black) | `#FFFFFF` |
| Accent / CTA / numerals / markers | Orange `#F26A1F` |
| Secondary text | `#5F6368` |
| Card/panel on dark | Charcoal `#2B2B2B` |
| Dividers | `#E5E7EB` |

## 5. Footer (content slides)
`Loan Factory — LO Development · <Program> · Internal use only · <#>`

## 6. Content rules
- Open with the outcome; end with CTA + next steps slide.
- Speaker-friendly: slides support the talk, don't replace it.
- Obey all language rules (LO / TERA / Netlify / approved Loan Factory users; no Apex Advisor; no hype/guarantees/manifestation/borrow-to-join).
- Keep Sales & Marketing decks visibly free + separate from paid coaching.

## 7. Production note
Generate via the **pptx** skill. Provide per-slide titles + bullets + speaker notes. Apply the palette above. Run the Pre-Send Brand Checklist.
