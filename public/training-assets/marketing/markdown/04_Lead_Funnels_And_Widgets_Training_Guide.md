## Source Digest
This guide shows how a loan officer can create an external website widget from the Loan Factory portal. The page explains the SDK script, where to place it on an external website, how to add a widget, how to choose a widget type, and how to copy the generated script.

## Step-by-Step Walkthrough

### Goal
Create a Loan Factory lead generation widget that can be embedded on an external website or landing page.

### Who This Is For
Loan Factory loan officers who have a personal website, team website, landing page, blog, or partner page and want a lead capture tool connected to their Loan Factory account.

### Before You Start
You need access to your Loan Factory portal and the external website where the widget will be embedded. If you do not manage that website yourself, send the generated script to the person who manages the site.

### Exact Navigation Path
Loan Factory portal > Marketing Features > Lead funnels & Widgets > Widgets - External websites.

### Steps
1. Open **Lead funnels & Widgets** from the Marketing Features menu.
2. Review the SDK script shown at the top of the page. This script is inserted into the external website `<head>` section.
3. Click **Add** to create a new widget record.
4. Enter the external website URL where the widget will live.
5. Add a short internal description so you know what the widget is for.
6. Choose the widget type. Available examples shown in the screenshots include Quote form, Long form, Long form 2, HomeReady First long form, Rate table, 1003 Application Widget, Today's rate, Rate chart, Veterans affairs loan, Home Equity Line of Credit, Home Equity Line of Credit Application, Basic form, and Qualification Mortgage Calculator.
7. Optional, enter a lead source so incoming leads can be tracked correctly.
8. Customize the widget if needed.
9. Copy the generated script and submit the widget record.
10. Place the widget script on the external website where you want the widget to appear.

## Common Mistakes
- Skipping the SDK script in the website `<head>` section.
- Choosing a widget type before knowing the landing page goal.
- Leaving lead source blank when tracking matters.
- Sending a script to a web developer without the external website URL and widget purpose.
- Assuming the widget is live before testing the external page.

## How to Know It Worked
The widget should display on the external website. When a visitor interacts with it, the lead source and widget source should make it clear where the lead came from.

## 90 Second Walkthrough Script
**Hook:** If you have an external website, do not just send people there and hope they call. Add a Loan Factory widget so visitors can raise their hand.

**Steps:** Go to Marketing Features, then Lead funnels & Widgets. Open the external websites widget tab. Review the SDK script, click Add, enter your external website URL, pick the widget type, enter a lead source, customize if needed, then copy the script and submit.

**Common Mistake:** Do not forget the SDK script in the website head. Without that, your widget may not load.

**Close:** After the widget is embedded, test it from the public page and confirm the lead source is tracking correctly.
