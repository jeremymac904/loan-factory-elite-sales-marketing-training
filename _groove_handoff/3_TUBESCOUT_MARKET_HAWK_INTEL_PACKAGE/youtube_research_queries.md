# YouTube Research Queries — TubeScout Bank

**Purpose:** Standardized search-query bank for the TubeScout researcher. Run each query in YouTube native search (sort: "This year" or "This month" for freshness; sort: "View count" for evergreen). Capture results into `youtube_research_capture_template.csv`.

**Rules:**
- Public videos only.
- Never copy scripts verbatim — paraphrase the takeaway.
- Cite URL + date_checked on every captured row.
- Mark videos with stale data (e.g., expired rate quotes, old comp grids) as "evergreen_y_n = N" and skip for AI Advantage clips.
- Loan Factory tone applies to anything we recycle: no hype, no public-facing chatbot framing, internal-only safety posture.

## Researcher Workflow

1. Pick a theme. Run 4-8 queries.
2. For each result that scores high on hook + evergreen value, capture into `youtube_research_capture_template.csv`.
3. For each captured row, name the `adapt_to` target (FaceGram seed / Scripts library / AI Advantage micro-lesson / Prompts library / Training Library / Coaching session / Team Leader Playbook / Marketing review).
4. Submit a weekly TubeScout digest to the strategy lead with: total reviewed, total captured, top 3 evergreen finds, top 3 "do not adapt" finds (and why).

---

## Cadence

- Daily quota: 20 videos reviewed, 5 captured.
- Weekly quota: 100 reviewed, 25 captured, 1 digest.
- Monthly: rotate 1 theme into deeper dive (full channel review for top 3 channels in that theme).
