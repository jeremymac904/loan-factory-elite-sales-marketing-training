# Loan Officer Support Walkthrough

**Audience:** Loan Factory loan officer support staff and the leaders who route requests to them.

**Goal:** Tour the inbound triage path: `/support-routing`, `/lender-escalation`, `/resources`, and the user lookup affordances available to a support role.

**Starting route:** `/admin/view-as` (master admin selects the Loan Officer Support role, then lands on `/`)

**Estimated runtime:** 4–6 minutes

**Suggested intro script:** "This is the Loan Officer Support view. I'm previewing it through View-As so you can see exactly how a support teammate triages inbound requests from approved Loan Factory loan officers."

## Exact steps

1. From `/admin/view-as`, pick the Loan Officer Support role, confirm the orange banner, and land on `/`.
2. Open `/support-routing` and walk the routing dashboard.
3. Open `/lender-escalation` and walk the escalation form fields without submitting.
4. Open `/resources` and scroll the resource library.
5. Use the user lookup affordance from the nav or `/support-routing` to find a seeded demo LO by name.
6. Return to `/support-routing` so the closing frame shows the triage hub.
7. Click **Exit View-As**. Land URL: `/admin/view-as`.

## What to click

- The Support nav entry on `/`.
- The user lookup search field inside `/support-routing` (type a seeded demo name only).
- The escalation form fields on `/lender-escalation` — visual walk only, do not submit.

## What success looks like

- Viewer can see all three core support routes load: `/support-routing`, `/lender-escalation`, `/resources`.
- Viewer can see the user lookup return a seeded demo LO.
- Viewer can see the orange View-As banner throughout.
- Viewer can confirm support does not see `/admin` or `/admin/users` in the nav.

## Common mistakes

- Submitting the lender escalation form during the recording.
- Typing a real LO name into the user lookup. Use seeded demo names only.
- Skipping `/resources` because it looks like a static page — it is part of the daily flow.

## Recording notes

- The orange View-As banner must remain visible.
- Use seeded demo data only; no real borrower or LO PII.
- Keep the cursor steady on the user lookup result so the viewer can read the matched record.

**Suggested closing script:** "That is the Loan Officer Support triage path. Exiting View-As and moving to the LO Mastery 249 member view."
